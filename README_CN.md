专为 ComfyUI 打造的高性能全能型自定义节点工具箱，旨在优化工作流逻辑、增强数据处理能力，并提供电影级的视觉控制工具。

---

## 🌟 核心特性
* **高级工作流优化：** 丝滑分流与管理复杂的逻辑链路。
* **电影级视觉工具：** 专为高保真渲染管线定制，精准控制色彩搭配与材质纹理留存。
* **高性能执行：** 优化后端处理机制，超低显存占用，内存释放干净利落。

## ⚙️ 安装指南

### 方法一：通过 ComfyUI Manager 安装（推荐）
1. 在 ComfyUI Manager 中搜索 `ComfyUI-FeiHou-Toolbox`。
2. 点击 **Install** 安装。
3. 重启 ComfyUI。

### 方法二：手动 Git 克隆
进入你的 ComfyUI `custom_nodes` 根目录下，打开终端运行：

**git clone https://github.com/FX-FeiHou/ComfyUI-FeiHou-Toolbox.git**

重启 ComfyUI 并硬刷新浏览器页面 (Ctrl + F5)。

* **[节点名称 2]** - 简述该节点的功能（例如：细腻提取材质纹理与光影微细节）。
